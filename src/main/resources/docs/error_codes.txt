# Error 400 Bad Request
This indicates the request body was malformed or missing required fields. Check the JSON syntax and field validation rules.

# Error 401 Unauthorized
The API token is missing, invalid, or has expired. You must refresh the token or log in again to generate a new one.

# Error 403 Forbidden
The user is authenticated but does not have permission to access the requested resource. Check the user's role and permissions.

# Error 500 Internal Server Error
An unexpected condition was encountered on the server. These errors are logged automatically; please contact support if the issue persists.
